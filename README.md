# OOP Car Inheritance Console Application

A C# console application created to demonstrate Object-Oriented Programming (OOP) concepts.

## Concepts Demonstrated
- Interfaces
- Abstract classes
- Inheritance
- Method overriding
- Encapsulation

## Structure
- ICar interface defines common car behaviors
- Car abstract class provides shared properties
- Hatchback and Sedan implement specific behaviors

## Technologies
- C#
- Console Application
- Visual Studio
